2. Backend lookup endpoints added
=================================

Added:
- GET /api/media/lookup?search=&type=&page=&pageSize=
- GET /api/users/lookup?search=&page=&pageSize=
- GET /api/admin/users/lookup?search=&page=&pageSize=
- GET /api/groups/lookup?search=&visibility=&page=&pageSize=
- GET /api/collections/lookup?search=&page=&pageSize=
- GET /api/moderation/scopes/lookup?scopeType=&search=&page=&pageSize=

Lookup behavior:
- Media lookup excludes deleted media, supports title search, optional type filtering, deterministic ordering, normalized pagination, and display-rich subtitles.
- Public user lookup searches username/display name only and does not expose email.
- Admin/moderator user lookup requires moderator policy and can search email without returning email in the DTO.
- Group lookup shows public groups to anonymous users and private groups only to members.
- Collection lookup shows public collections to anonymous users and follower/private-visible collections only to allowed users.
- Moderation scope lookup requires admin policy and returns unified Group/Collection/Media/Global picker options.

3. Lookup tests added
=====================

Added backend/RateOple.Api.Tests/Lookup/LookupContractTests.cs.

Covered:
- Deleted media are excluded from media lookup.
- Lookup pagination normalizes invalid page/pageSize values.

- Moderator user lookup can search email while still returning privacy-safe DTOs.
- Private groups are hidden from anonymous users and visible to members.
- Follower-only collections are hidden from anonymous users and visible to followers.
